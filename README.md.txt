# 🛒 Sistema de Loja em Python

Este é um projeto simples de um sistema de loja feito em Python utilizando o Google Colab.

O sistema possui funcionalidades básicas como:
- Exibição de produtos
- Carrinho de compras
- Cálculo de total
- Finalização da compra

---

## ▶️ Como usar

Abra o notebook `sistema_loja.ipynb` no [Google Colab](https://colab.research.google.com/) ou em um Jupyter Notebook.
